### Current Projects

**[adaptive-encoding-anomaly-detector](https://github.com/bumblB33/adaptive-encoding-anomaly-detector)**
Categorical feature-encoding selection as a Mixed-Integer Linear Program.
Benchmarks the selected encoding against a one-hot baseline using
Local Outlier Factor on the IEEE-CIS fraud dataset. The
MILP's decision matrix doubles as an explanation of encoding choices. 

**[Radio_Meteor_Zoo](https://github.com/bumblB33/Radio_Meteor_Zoo)**
Radio Meteor Zoo is part of the [Zooniverse](https://www.zooniverse.org/) citizen science initiative. The goal was to crowd-source meteoroid research using radio images to detect and study the meteoroid population entering the Earth's atmosphere. Playing around with computer vision to the same effect. :) Applying ML clustering and dimensionality reduction (Gaussian Mixture Models, t-SNE, UMAP) to classify meteor scatter radio signals from the Geminids shower. 

**[address-space-reverse-engineering](https://github.com/bumblB33/address-space-reverse-engineering)**
Research whether an unprivileged process on Windows can infer another
process's memory layout through same-host observations. The goal is to see if it's possible to partially circumvent ASLR without privileged access. Borrowed the idea from my optimization algorithm work - just applied to DLL addressing. 

**[selective-forecasting-benchmark](https://github.com/bumblB33/selective-forecasting-benchmark)**
Benchmarks to evaluate how well NIXTLA's open-source forecasting models can predict multi-location business sales.
